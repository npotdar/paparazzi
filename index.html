A quick guide to using git for our project

Always start any procedure with 1. Open terminal 2. Type cd ./paparazzi

Initial Setup

Change username

  1. Type git config --global user.name "firstname" replacing firstname with your name
  2. Confirm the username by typing git config --global user.name

Add our repository

  1. Type git remove rm roland to remove the roland repository
  2. Type git remote add origin https://github.com/npotdar/paparazzi.git to add repository
  3. Verify settings by git remote -v to check the origin respository is there
  4. Type git checkout mavlabcourse4 to move to the mavlabcourse4 branch

Pulling our repository (updating your local version from online)

  1. Type git pull origin mavlabcourse4

Pushing and committing changes to repository (updating online from local version)

  1. Type git status lets you check which files will be updated online
  2. Type git add . (notice the period!) to add all your updated files to the update
  3. Type git commit -m "your message" to prepare addition and replace your message with a descriptive message of the changes
  4. Type git push origin mavlabcourse4 to send changes online
